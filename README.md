# Full Stack Development (FSD) Labs

This repository contains the laboratory experiments completed as part of the **Full Stack Development (FSD)** course during the **4th Semester of Engineering**.

Each lab demonstrates different concepts related to front-end development, Progressive Web Applications (PWA), and modern web technologies.

---

## Repository Structure

```
FSD-Labs
│
├── lab1
├── lab2
├── lab3
├── lab4
├── lab5
└── lab6
```

Each folder contains the source code and files related to the respective lab experiment.

---

## List of Experiments

### Lab 1 – HTML Fundamentals

* Introduction to HTML
* Creating basic webpage structure
* Using headings, paragraphs, lists, and links

### Lab 2 – CSS Styling

* Applying CSS styles to webpages
* Using selectors and properties
* Improving webpage layout and design

### Lab 3 – JavaScript Basics

* Introduction to JavaScript
* DOM manipulation
* Event handling

### Lab 4 – Responsive Web Design

* Designing responsive layouts
* Using CSS media queries
* Making webpages compatible with different screen sizes

### Lab 5 – Progressive Web App (PWA)

* Creating a PWA manifest
* Service Worker registration
* Offline capability implementation

### Lab 6 – Push Notifications

* Implementing web push notifications
* Notification permissions
* Displaying and managing notifications

---

## Technologies Used

* HTML5
* CSS3
* JavaScript
* Progressive Web App (PWA) technologies
* Service Workers
* Web Notifications API

```

```

3. Open the HTML file in a browser

```
index.html
```
